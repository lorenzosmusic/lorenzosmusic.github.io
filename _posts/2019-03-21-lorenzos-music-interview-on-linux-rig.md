After our article in <a href="https://www.lorenzosmusic.com/2018/09/lorenzos-music-in-forbes-using-open.html">Forbes about being an open source band</a>, we've been getting a lot of love from the open source and Linux community. Which is awesome!<br />
<br />
I was contacted by <b>Linux Rig</b> to talk about the Linux set up&nbsp;we use and also about how we started using open source software to create and record.<br />
<blockquote class="tr_bq">
<div style="text-align: right;">
I used to use a bunch of proprietary software for artwork and I heard about this operating system that had all of this open source software built into it and I was curious. So I tried it out.<br />
<a href="https://linuxrig.com/2019/03/20/the-linux-setup-tom-ray-lorenzos-music/" target="_blank">- Lorenzo's Music on linuxrig.com</a></div>
</blockquote>
