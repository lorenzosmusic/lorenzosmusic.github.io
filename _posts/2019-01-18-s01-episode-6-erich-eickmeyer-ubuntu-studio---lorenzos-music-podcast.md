In this episode, I get to meet&nbsp;<b>Erich Eickmeyer</b>&nbsp;Council Chair at&nbsp;<b>Ubuntu Studio</b>.<br />
<br />
When we recorded our "<a href="https://www.lorenzosmusic.com/2018/08/rom-comm-mixtape.html" target="_blank">Rom-Comm Mixtape</a>" release we decided to try an do the whole thing using only open source tools and software. We even got a <a href="https://www.lorenzosmusic.com/2018/09/lorenzos-music-in-forbes-using-open.html" target="_blank">write up in Forbes because of it</a>.<br />
<br />
We ended&nbsp;up using a Linux distribution called <b>Ubuntu Studio</b>. We chose it because it's an entire computer&nbsp;operating&nbsp;system that comes preloaded with tons of free software&nbsp;for musicians, artists &amp; filmmakers.<br />
<br />
I reached out to the Ubuntu Studio Facebook page to see if anyone would be interested in talking about the project on the podcast because I wanted to support them for what they've done.<br />
<br />
Check out Ubuntu Studio for yourself at <a href="https://ubuntustudio.org/" target="_blank">https://ubuntustudio.org</a><br />
