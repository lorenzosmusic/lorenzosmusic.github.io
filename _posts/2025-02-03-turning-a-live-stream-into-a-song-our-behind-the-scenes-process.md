{% include video-embed.html video_id="HRz64CJ7iw4" title="Turning a Live Stream into a Song: Our Behind-the-Scenes Process" %}<p style="text-align: left;">You might remember that <a href="#">I mentioned turning one of our live streams into a new song</a>. This post kicks off the behind-the-scenes look at how we’re doing it, and if you’re into tech, recording, or just curious about the creative process, you’re in the right place.</p><h2 style="text-align: left;">Step 1: Downloading the Live Stream Audio </h2><p style="text-align: left;">The first thing I needed to do was download the live stream from our YouTube channel. We can download it directly from our YouTube Studio account. Once that’s done, I need to pull the audio from the video, which I discovered a super simple way to do. </p><p style="text-align: left;">I used&nbsp;<b>Audacity</b>, an open-source audio editor, on my Ubuntu Studio setup. Here’s the cool part: I can just drag the video file into Audacity, and it automatically extracts the audio from the video into the software. From there, I can export it as a WAV file to use for working on the song. </p><h2 style="text-align: left;">Step 2: Why We Use Ubuntu Studio </h2><p style="text-align: left;">Now, if you’re wondering what <a href="https://ubuntustudio.org/" target="_blank">Ubuntu Studio</a> is, let me break it down. It’s a Linux-based operating system specifically designed for creators. It comes preloaded with everything from recording and mixing software to photo editing and video tools—all open-source and free to use.&nbsp;</p><p style="text-align: left;">A few years ago, <a href="https://www.lorenzosmusic.com/2018/09/lorenzos-music-in-forbes-using-open.html">we decided to fully embrace open-source tools as a band</a>, and Ubuntu Studio has been key to that decision. It allows us to have a complete production setup without expensive licenses, which is why we’ve stuck with it for years. </p><h2 style="text-align: left;">Step 3: Recording with Ardour </h2><p style="text-align: left;">For this song, I’m using <a href="https://ardour.org/" target="_blank">Ardour, an open-source multitrack recording DAW (Digital Audio Workstation)</a> that’s comparable to industry tools like Pro Tools. Ardour is our go-to for recording everything from scratch tracks to fully polished songs. It’s loaded with professional features, yet because it’s open-source, it fits right into our philosophy of creativity. </p><p style="text-align: left;">This new song we are going to be working on is experimental, and I’m excited to see how it takes shape using the live stream as a foundation. Ardour is what I'll be using to layer new sounds and develop the track. </p><h2 style="text-align: left;">Step 4: Collaborating with GitHub </h2><p style="text-align: left;">Here’s where things get even more interesting. To manage the collaboration on this project with the rest of the band, we’re using <a href="https://github.com/lorenzosmusic" rel="nofollow" target="_blank">GitHub</a>.&nbsp;</p><p style="text-align: left;">Now, GitHub is traditionally used by software developers, but <a href="https://www.lorenzosmusic.com/2024/01/github-for-music-how-we-record-remotely.html">we’ve come up with our own way of adapting it for our music projects</a>. Using GitHub, we can collaborate on full Ardour sessions—sharing DAW files, effects, plugins, and mixing in real-time across locations. </p><p style="text-align: left;">This approach gives us flexibility and creative control, allowing each member of the band to contribute to the project. It’s a really powerful workflow, and I’m excited to show you more as we continue creating this track. </p><h2 style="text-align: left;">What’s Next? </h2><p style="text-align: left;">Now that I’ve got everything set up—the DAW session, the audio from the live stream, and our GitHub all set up—I can dive into creating the song.&nbsp;</p><p style="text-align: left;">I’ll be sharing more updates as the track takes shape, including more information about the tech and tools we'll be using. </p><p style="text-align: left;">If you’re interested in how we make music or curious about the tech behind our process, I’ll be breaking it down step-by-step, and if you want to learn more about how you can use these tools yourself, sign up for our email list to get updates on what’s coming next. </p><div id="mc_embed_shell">
